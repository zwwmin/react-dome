# react-dome
Vite+ React18+react-router-dom

This starter template includes:

- [Vite 3](https://vitejs.dev/guide/)
- [React18](https://react.docschina.org/)
- [React Router dom 6](https://reactrouter.com/en/main)


### Getting Started

1. git clone

2. yarn install

3. yarn dev

4. Packing project
